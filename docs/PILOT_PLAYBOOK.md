# Pilot Playbook (10-20 Families)

## Goal

Validate whether NurseApp improves family visibility and coordination speed in real weekly care operations.

## Pilot Profile

- Family caregivers managing 1-3 recipients.
- At least one collaborating caregiver per household.
- Existing routine currently managed via paper/text/spreadsheets.

## 2-Week Pilot Flow

1. Day 0 onboarding (30-45 min)
- Set role accounts.
- Add recipients.
- Import first daily routine.
- Configure escalation timings.

2. Week 1 monitoring
- Track missed tasks and acknowledgment speed.
- Confirm family can see done/missed in timeline.
- Gather friction notes from chat/import usage.

3. Week 2 optimization
- Tune categories and alert settings.
- Review KPI panel trends.
- Capture retention and referral intent.

## Success Metrics

- >= 70% daily active pilot households.
- >= 20% improvement in on-time completion vs baseline.
- < 20 min average acknowledgment delay.
- >= 60% pilot households continue into month 2.

## Interview Prompts (Weekly)

- What remained hard this week?
- Which alert felt noisy or missing?
- Did family visibility reduce coordination stress?
- What blocked daily use?

## Exit Decision

- Promote to paid early access when KPI thresholds are met and major blockers are resolved.
